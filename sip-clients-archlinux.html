Клиенты IP-телефонии под Linux

IP-телефон (он же софтфон) — программа, которая позволяет совершать исходящие или входящие звонки с компьютера, используя гарнитуру (наушники и микрофон). Для доступа к услугам IP-телефонии необходимо получить настройки поставщика услуг и оплатить услугу (то есть звонки согласно тарифам). В бизнесе IP телефония широко используется для организации горячих линий. IP телефония стала одним из ключевых аспектов для организации call-центров. Постепенно этот вид услуг начинает получать популярность среди малого и среднего бизнеса, и даже среди самых обычных пользователей, поскольку имеет определенные преимущества перед обычной связью. Нужно отметить, что клиентская программа - софтфон - это аналог физической трубки устройства, которое так и называется в продаже - IP-телефон. Для приема звонков и их совершения достаточно настроить клиент, следуя указаниям поставщика IP услуг и справке по клиентской программе. Серверные же решения служат для организации так называемых виртуальных АТС и обычному пользователю не нужны. Например, бизнес использует на клиентских машинах приложение Citrix Workspace App for Linux ( yay -S icaclient ), в котором реализована поддержка софтофонов[1] Бытует мнение, что IP-телефония — прерогатива пользователей ОС Windows. На самом деле сервера для IP телефонии часто обслуживаются именно на Unix-подобных операционных системах, и соответственно под них были написаны первые софтофоны. Перечислим популярные решения. 1.
