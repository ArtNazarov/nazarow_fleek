Скрипт настройки Arch Linux

Заметка посвящена возможностям скрипта, с помощью которого можно донастроить Arch сразу после установки Страница скрипта https://github.com/ArtNazarov/arch-postinstall Переходим в терминал, делаем скрипт запускаемым chmod +x arch-postinstall.sh Запускаем ./arch-postinstall.sh От предлагаемых возможностей можно отказаться, нажав N или Enter, чтобы установить выбираем Y

Возможности

Начальная настройка

  • Обновление ключей
  • Системные утилиты
  • Общее обновление системы

Установка ядер на выбор

  • Установка ядра Xanmod, или
  • Установка ядра Linux Tkg
  • Установка ядра Zen
  • Установка утилит сборки, компоновки, линковки, компиляции
  • Настройка сети
  • Утилиты управления частотой процессора

Пакетные менеджеры

  • Установка Flatpak
  • Установка Snap

    Настройка видеосистемы

  • Установка Mesa
  • Установка Vulkan
  • Установка DBus broker для видео
 

При переустановке

  • Очистка кеша шрифтов
  • Очистка предыдущей инсталляции Chrome

    Безопасность

  • Установка утилит безопасности (apparmor, firejail)

    Звуковая система и bluetooth

  • Установка утилит для bluetooth
  • Установка звуковой системы (Alsa, Pipewire или PulseAudio)
  • Установка аудиоплееров

    Прочие приложения (необязательно)

  • Установка торрент клиента
  • Инструменты разработчика
  • Установка софта из flatpak
  • Установка центра приложений pamac
  • Установка видеоплеера
  • Установка хранилки паролей
  • Установка WINE для приложений Windows

    Оптимизации

  • Утилита ANANICY
  • Утилита RNG (не ставить, если выбран ANANICY)
  • Утилита HAVEGED
  • Настройка Trim для ssd
