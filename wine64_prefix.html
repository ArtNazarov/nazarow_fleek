Настройка 64 битного префикса wine для запуска программ Windows

Статья освещает настройку 64 битного префикса. Операционная система blue star archlinux Убедимся, что пакет wine доступен sudo pacman -Fy wine Поиск также можно выполнить командой sudo pacman -Ss wine | less

    1. Устанавливаем cabextract sudo pacman -Sy cabextract
    2. Устанавливаем wine, wine-mono, wine-gecko и winetricks sudo pacman -Sy wine sudo pacman -Sy wine-mono wine-gecko sudo pacman -Sy winetricks
    3. Скачиваем файл и запускаем установку компонент ./wt-install-64.sh
  1. Отредактируйте переменные среды своего терминала. В Konsole это Настройка / Настроить профиль / Переменные окружения. Добавьте строки и измените username на имя пользователя WINEDEBUG=-all arch=64 WINEPREFIX=/home/username/.wine64
  2. Прописываем переменные в .bashrc nano .bashrc добавляем export arch=64 export WINEPREFIX=/home/username/.wine64 export WINEDEBUG=-all сохраняем ctrl+o, ctrl+x, применяем изменения source .bashrc
  3. Дополнительные библиотеки можно переписать с зеркала по ссылке, запустив файл inst-from-mirror.sh
