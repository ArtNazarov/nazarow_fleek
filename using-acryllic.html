Блокируем рекламу и вредоносные сайты с помощью Acryliс в Windows

Acrylic - утилита DNS прокси, которая имеет много полезных функций. Одной из особенностей Acrylic является поддержка больших файлов hosts. В Windows, в том числе в 11 версии, имеется проблема, при которой служба DNS Cache начинает нагружать процессор, что приводит к заметному ухудшению общей производительности системы. Это известная проблема, которая до сих пор не решена. Для чего нужно использовать hosts файлы: блокировка телеметрий, блокировка вредоносных доменов, блокировка интернет-рекламы и так далее. Откуда можно взять hosts файл с большим списком сайтов? 1. https://raw.githubusercontent.com/StevenBlack/hosts/master/hosts Сводный список, блокирующий рекламу и вредоносы. 2. https://pgl.yoyo.org/adservers/serverlist.php?showintro=0;hostformat=hosts режет только рекламу 3. https://www.encrypt-the-planet.com/downloads/hosts разнообразная телеметрия Порядок настройки Acrylic 1. Скачиваем дистрибутив https://sourceforge.net/projects/acrylic/files/Acrylic/2.1.0/Acrylic.exe/download 2. Устанавливаем 3. Открываем Acrylic UI, нажимаем File -> Open Acrylic hosts. Вставляем строки из hosts файлов, перечисленных выше. 4. Нажимаем File -> Save 5. Программа запросит подтверждение перезапуска службы, соглашаемся 6. Открываем файл конфигурации File -> Open Acrylic Configuration. Находим строку PrimaryServerAddress, меняем на PrimaryServerAddress=77.88.8.88 Находим строку SecondaryServerAddress, меняем на SecondaryServerAddress=77.88.8.2 Тем самым мы прописываем безопасный DNS от Яндекса ( подробнее на https://dns.yandex.ru/), можно прописать семейный DNS, если хотите отключить сайты для взрослых Сохраняем (Ctrl+S), перезапускаем. Альтернатива - прописать 8.26.56.26 и 8.20.247.20 - безопасные DNS от Comodo 7. Остается прописать адрес прокси-сервера 127.0.0.1 в свойствах подключения. Рассмотрим на примере Windows 11 Win+R, control, Сети и Интернет, Центр управления сетями, Изменение параметров адаптера, Беспроводная сеть, Правый щелчок мышью, Свойства, выбираем IP версии 4, кнопка свойства, переключаем на использовать следующие адреса DNS, вводим 127 0 0 1 в пункте Предпочитаемый DNS сервер. Также можно прописать и другим способом - правый щелчок по значку Wi-Fi в правом нижнем углу, Параметры сети и интернета, Wi-Fi, Свойства (имя беспроводной сети), Изменить параметры DNS, убираем автоматически, переключаем на вручную, включаем IPV4, прописываем предпочтительный DNS сервер 127.0.0.1 8. Обновление DNS Cache - Win+R, cmd, набираем команду ipconfig /flushdns, закрываем. На этом настройка Acrylic в основном завершена. В окне Acrylic UI должно быть написано Service Installed & Running - служба установлена и запущена. Если необходимо перезапустить службу вручную, выбираем Actions -> Restart Acrylic Service.
