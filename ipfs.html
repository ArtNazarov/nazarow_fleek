IPFS

Чтобы опубликовать сайт в децентрализованной сети IPFS

  1. Создайте аккаунт https://app.fleek.co/ с помощью Github
  2. Создайте новый проект на github и добавьте туда файлы сборки
  3. Нажмите add new site в разделе hosting
  4. Выберите проект в списке
  5. Оставьте тип сборки Other
  6. Дождитесь окончания развертывания
  7. Имя хоста вида sitename.on.fleek.co можно поменять в настройках позднее.

Примечания - каждое новое развертывание происходит в сетевой узел вида https://ipfs.fleek.co/ipfs/Qmbb28WDFnt7PxE3cmQna2AsZ92TH1Mc6vjaUDgGhZVyLs , где Qmbb28WDFnt7PxE3cmQna2AsZ92TH1Mc6vjaUDgGhZVyLs - хеш очередной сборки

Узнать текущий хеш можно с помощью пункта verify on ipfs

До публикации хеш не рассчитывается заранее и не может быть известен генератору, поэтому следует использовать домен, выданный на fleek

.

Тем не менее сборка будет доступен по хешу, например, https://ipfs.io/ipfs/Qmbb28WDFnt7PxE3cmQna2AsZ92TH1Mc6vjaUDgGhZVyLs/
