Перенос кеша ArchLinux на другой раздел

Файлы кеша могут занимать много места на корневом разделе. Рассмотрим как изменить расположение кеша Переместим кеш в новое расположение sudo mv .cache /some/new/path Изменим права на файлы sudo chown username:username /some/new/path -R Сотрем кеш по фактическом расположению sudo rm .cache -rf Сделаем символическую мягкую ссылку с именем .cache, которая укажет на другой раздел ln -s /some/new/path/.cache .cache Проверим, что ссылка указывает на новое расположение ls .cache -la выведет .cache -> /some/new/path/.cache Аналогичным образом можно переместить кеш не только из домашнего каталога пользователя ( ~/.cache ), но и из папки /var/cache
