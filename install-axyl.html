Установка Axyl Linux

Axyl Linux - вариант archlinux с установкой различных графических оболочек с инсталлятором archinstall Рассмотрим шаги установки в виртуальной машине. После установки система займет 3 Гб на диске (при создании машины использовался vdi) Нажимаем Win + Enter, чтобы попасть в терминал ( Alt + Enter ) Запускаем от имени администратора archinstall. Выбираем язык. Для первичной настройки лучше оставить us Дожидаемся проверки доступности зеркал Выбираем Russia или 47. Можно выбрать Germany Выбираем основной диск ( 1 ) Нажимаем enter и выбираем 0 - автоматическая разметка Выбираем тип файловой системы btrfs Выбираем Y чтобы автоматически создать подразделы Нажимаем enter, чтобы не шифровать диск Отказываемся от swap в оперативной памяти Выбираем имя компьютера в сети Вводим пароль пользователя root (суперпользователь) Повторно вводим для подтверждения Аналогично создаем дополнительного пользователя Можем добавить в группу sudoers для удобства настройки Выбираем тип системы - desktop для установки окружения рабочего стола Выбираем минималистичное i3 Выбираем конфигурацию окружения Выбираем открытые видео драйверы 1 Выбираем звуковую систему   Тип ядра , можно нажать enter для стандартного Перечисляем через пробел дополнительный софт Копируем настройки сети 0 На следующем экране выбираем часовой пояс Устанавливаем точное время Нажимаем Enter чтобы начать установку. На реальной машине форматирование приведет к потере данных, будьте внимательны.     Стартует форматирование Устанавливаются пакеты И так далее. Вы можете зайте в настроенную систему, нажав Y (chroot) После установки вводим reboot для перезагрузки. В virtualbox изымаем виртуальный образ из дисковода Enter Вводим свой логин В очередном окне выбираем клавишу-модификатор для i3 Нажимаем win + enter - запускается терминал. вводим mc + enter Запустился mc, указанный как дополнительный пакет
