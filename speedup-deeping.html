Решаем проблему со скоростью в Deepin Linux

Скачивание из официального репозитория медленное. Чтобы решить проблему нужно поменять зеркало на российское в файле /etc/apt/sources.list. Для этого с правами суперпользователя достаточно закомментировать официальный репозиторий и добавить корректную строку, ссылающуюся на зеркало Яндекса. После правки выполнить sudo apt-get update в терминале.
