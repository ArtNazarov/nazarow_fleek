Устранение ошибки kdfontop

В случае с ошибкой KDFONTOP не загружается локализация консоли Чтобы устранить ошибку

    1. Редактируем переменную HOOKS в файле sudo nano /etc/mkinitcpio.conf
    2. Удаляем consolefont, keymap, пишем sd-console. Примерный вид HOOKS="base udev autodetect modconf block keyboard sd-vconsole filesystems fsck"
    3. От рута делаем пересборку инита 
      # mkinitcpio -P
    4. В .bashrc добавляем команду setfont cyr-sun16
    Примечание (должны быть выполнены условия): Содержимое /etc/vconsole.conf KEYMAP=ru FONT=cyr-sun16 CONSOLEFONT="cyr-sun16" CONSOLEMAP="" Должен быть доступен шрифт из kbd ls /usr/share/kbd/consolefonts/ | grep 'cyr-' | cut -d. -f1 В /etc/locale.gen должна быть раскомментирована ru_RU.UTF-8 UTF-8 При необходимости раскомментировать, перегенерировать и назначить переключение: 
    # locale-gen
    # localectl set-locale ru_RU.UTF-8
    # localectl set-x11-keymap --no-convert us,ru pc105 "" grp:alt_shift_toggle
