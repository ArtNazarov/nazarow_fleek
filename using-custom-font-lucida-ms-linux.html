Решение проблемы со шрифтом Lucida Grande

В ряде случае веб-сайты могут отображаться неправильно из-за того, что сайт использует нестандартный шрифт Веб-мастера игнорируют так называемые безопасные веб-шрифты (которые имеются во всех основных операционных системах). Рассмотрим пример В данном случае сайт сокращателя ссылок Яндекса использует шрифт Lucida Grande. Также можно отметить, что есть указание на шрифты MS, которые разумеется отсутствуют в системе (автор использует Linux) Обратим внимание, что если дизайнер не задал жестко font, то браузер использует шрифты из настроек системы и проблема пропадает сама собой. Но править код сайта мы не можем (точнее, мы могли бы поставить заплатку с помощью расширения типа Stylish, но это не решит проблему глобально) Таким образом, чтобы исправить проблемы с отображением шрифтов нужно а) добавить корректный шрифт в систему б) доустановить шрифты MS. Шрифт Lucida (в архиве) можно взять отсюда https://fonts-online.ru/fonts/lucida-grande Если в системе уже есть шрифт ( файл может иметь другое имя), удаляем его. Шрифты MS можно получить следующим способом, если установлена виртуальная машина - зайти в папку C:\Windows\Fonts и скопировать на папку основного компьютера файлы ttf. Затем полученные файлы нужно скопировать (например, с помощью mc) в папку ~/.fonts Далее следует закрыть программу (браузер, текстовый редактор) и открыть снова, чтобы произошло перечитывание доступных шрифтов.
