Команды flatpak и snap в сравнении

Варианты команд для flatpak и snap

Назначение Flatpak Snap
Установка пакета flatpak install аргумент 1) источник id-приложения 2) путь к локальному файлу .flatpakref 3) короткое имя пакета 4) ссылка на файл в сети .flatpakref snap install имя
Просмотр установленных flatpak list —app snap list
Обновить установленный flatpak update id-приложения snap refresh имя
откатить версию flatpak remote-info --log flathub id-приложения flatpak update --commit=<COMMIT_ID> flatpak mask id-приложения snap revert имя
удалить пакет flatpak uninstall <Application ID> или удаление неиспользуемого flatpak uninstall —unused snap remove имя
Поиск пакета flatpak search <искомая фраза>
Запуск flatpak run <Application ID> приложение будет доступно по пути /snap/bin
Устранение неисправностей flatpak repair
Создание снимка snap save
Восстановление снимка snap restore
