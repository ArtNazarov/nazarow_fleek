Сравнение команд для различных пакетных менеджеров

Перечень команд для управления установкой, обновлением, удалением и восстановлением программ в Linux для различных пакетных систем

Основные операции

Действия Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Поиск пакета pacman -Ss dnf search apt search zypper search или zypper se [-s] emerge -S
Установка пакета по имени pacman -S dnf install apt install zypper install или zypper in emerge [-a]
Обновление пакетов до новых версий pacman -Syu dnf upgrade apt update и затем apt upgrade zypper update или zypper up emerge -[a]uDN @world
Полное обновление системы pacman -Syu dnf distro-sync apt update и затем apt dist-upgrade zypper dup emerge -[a]uDN @world
Удаление пакета со всеми его зависимостями pacman -Rs dnf remove apt autoremove zypper remove или zypper rm emerge -[a]vc
Удаление пакета с конфигурационными файлами pacman -Rn ? apt purge ? ?
Удаление пакета с зависимостями и с конфигурацией pacman -Rns ? apt autoremove --purge ? ?
Удаление зависимостей, которые более не нужны ( осиротевшие ) pacman -Qdtq | pacman -Rs - (-Qdttq для игнорирования необязательных ) dnf autoremove apt autoremove zypper rm -u (только удаляет пакет) либо zypper packages --unneeded (выдает список) emerge [-a] --depclean
Удаление пакетов, не входящих ни в один репозиторий pacman -Qmq | pacman -Rs - dnf repoquery --extras aptitude purge '/home/username/o'
Установленный ранее пакет отметить как явно требуемую зависимость pacman -D --asexplicit dnf mark install apt-mark manual zypper install --force (обходной вариант, потребует переустановки пакета) emerge --select
Установка пакета как зависимости, но не явно требуемой pacman -S --asdeps dnf install и затем dnf mark remove apt-mark auto n/a emerge [-a] --oneshot или emerge [-a] -1
Только скачивание пакетов, без распаковки и установки pacman -Sw dnf download apt install --download-only (в кеш пакетов) или apt download (в обход кеша пакетов) zypper --download-only emerge [-a] --fetchonly
Очищает локальные кеши pacman -Sc или pacman -Scc dnf clean all apt autoclean удалит только ненужные или apt clean zypper clean eclean distfiles
Оболочка для ввода команд dnf shell zypper shell
Показать журнал действий открыть /var/log/pacman.log dnf history открыть /var/log/dpkg.log открыть /var/log/zypp/history открыть /var/log/portage
Получение дампа системной информации, состояния пакетов просмотр /var/lib/pacman/local просмотр /var/lib/rpm/Packages apt-cache stats emerge --info
Доставить изменения по e-mail apt install apt-listchanges

Запрос определенных пакетов

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Вся или большая часть информации по пакету pacman -Si или pacman -Qi dnf list или dnf info apt show или apt-cache policy zypper info или zypper if emerge -S, emerge -pv или eix
Отображение информации о локальном пакете pacman -Qi rpm -qi / dnf info installed dpkg -s или aptitude show zypper info или rpm -qi emerge -pv или emerge -S
Отображение информации по удаленному пакету pacman -Si dnf info apt-cache show или aptitude show zypper info emerge -pv и emerge -S или equery meta
Показать файлы, предоставленные локальным пакетом pacman -Ql rpm -ql dpkg -L rpm -ql equery files либо qlist
Показать файлы, предоставленные удаленны пакетом pacman -Fl dnf repoquery -l или repoquery -l (из пакета yum-utils) apt-file list pfl
Запрос пакета, который содержит нужный файл pacman -Qo rpm -qf (только установленные) or dnf provides (всё) or repoquery -f (из пакета yum-utils) dpkg -S либо dlocate rpm -qf (только установленные) или zypper search -f (всё) equery belongs либо qfile
Перечислить файлы в пакете pacman -Ql или pacman -Fl dnf repoquery -l dpkg-query -L rpm -ql equery files или qlist
Поискать с помощью регулярного выражения pacman -F dnf provides apt-file search zypper what-provides либо zypper wp equery belongs (только установленные) либо pfl
Выполнить поиск, чтобы найти пакет, содержащий файл pacman -F dnf provides apt-file search или auto-apt zypper search -f equery belongs или qfile
Показать историю изменений файла pacman -Qc dnf changelog apt-get changelog rpm -q --changelog equery changes -f

Запрос списков пакетов

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Поиск пакетов по имени, описанию pacman -Ss dnf search apt search zypper search или zypper se -s emerge -S или eix
Список пакетов, которые можно обновить pacman -Qu dnf list updates or dnf check-update apt list --upgradable zypper list-updates или zypper patch-check (just for patches) emerge -uDNp @world
Показать список всех пакетов pacman -Sl dnf list available apt-cache dumpavail либо apt-cache dump (только кеш) или apt-cache pkgnames zypper packages portageq all_best_visible /
Список установленных пакетов pacman -Q dnf list installed dpkg --list | grep ^i zypper search --installed-only qlist -IC
Список пакетов установленных, но недоступных в источниках pacman -Qm dnf list extras apt --installed list | grep ,local zypper se -si | grep 'System Packages' eix-test-obsolete
Список недавно добавленных в источник пакетов dnf list recent aptitude search '/home/username/N' или aptitude forget-new eix-diff
Список установленных локально pacman -Q rpm -qa dpkg -l или apt list --installed zypper search -s или rpm -qa qlist -ICv
Поиск по локальным пакетам по названию и описанию pacman -Qs rpm -qa '*<str>*' aptitude search '/home/username/i(/home/username/n $name|/home/username/d $description)' eix -S -I
Список пакетов, не требуемых никаким другим pacman -Qtt dnf leaves или package-cleanup --leaves --all deborphan -anp1 emerge -pc
Список пакетов установленных явно не как зависимости pacman -Qe dnf history userinstalled apt-mark showmanual grep -E '^i\+' (обход) emerge -pvO @selected или eix --selected
Список пакетов установленных автоматически как зависимости pacman -Qd grep -E '^i[^+]' (обход) apt-mark showauto

Запрос зависимостей пакета

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Отображение обратных зависимостей ( пакетов, для которых требуется установка X) pacman -Sii либо pacman -Qii dnf repoquery --alldeps --whatrequires либо repoquery --whatrequires apt-cache rdepends или aptitude search /home/username/D$pattern zypper search --requires emerge -pvc
Показать конфликтующие пакеты с данным pacman -Si или pacman -Qi dnf repoquery --conflicts aptitude search '/home/username/C$pattern'
Перечислить все прямые зависимости для данного pacman -Si или pacman -Qi dnf repoquery --requires или repoquery -R apt-cache depends или apt-cache show zypper info --requires emerge -ep
Посмотреть, что предоставляет данный пакет pacman -Sii или pacman -Qii dnf repoquery --provides dpkg -s или aptitude show zypper info --provides equery files or qlist
Список всех пакетов для которых требуется данный pacman -Sii dnf repoquery --installed --alldeps --whatrequires aptitude search /home/username/D{depends,recommends,suggests}:$pattern или aptitude why zypper search --requires equery depends -a
Показать устаревшие pacman -Si или pacman -Qi dnf list obsoletes apt-cache show
Создать выходный данные, пригодные для обработки в dotty apt-cache dotty

Управление источниками установки

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Настройки правка /etc/pacman.conf правка /etc/yum.repos.d/${REPO}.repo правка /etc/apt/sources.list правка /etc/zypp/repos.d/${REPO}.repo layman или eselect repository
Добавление источника правка /etc/pacman.conf /etc/yum.repos.d/*.repo apt-cdrom add zypper service-add layman или overlays
Обновить информацию по источнику pacman -Sy dnf clean expire-cacheа затем dnf check-update apt-get update zypper refresh или zypper ref emerge --sync или layman -S
Вывести все источники установки cat /etc/pacman.d/mirrorlist cat /etc/yum.repos.d/* apt-cache policy zypper service-list layman -l или eselect repository list
Вывести все пакеты , доступные в репозитории paclist <repo> eix --in-overlay
Сделать источник недоступным dnf --disablerepo= emerge package::repo-to-use
Загрузить пакеты из другой версии дистрибутива pacman -S repo_name/package dnf --releasever= apt-get install -t release package или apt-get install package/release (не охватывает зависимости) echo "category/package /home/username/amd64" >> /etc/portage/package.keywords а потомemerge package

Переопределения

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Заблокировать изменения пакета edit /etc/pacman.conf измени IgnorePkg edit dnf.conf добавь exclude apt-mark hold pkg zypper al or положи имя пакета в/etc/zypp/locks /etc/portage/package.mask
Удалить правило блокировки пакета edit /etc/pacman.conf удалит пакет из строки IgnorePkg apt-mark unhold pkg zypper rl or удали имя пакета из/etc/zypp/locks /etc/portage/package.mask (или package.unmask)
Показать список правил cat /etc/pacman.conf /etc/apt/preferences zypper ll или просмотр/etc/zypp/locks cat /etc/portage/package.mask
Установить приоритет пакета правка /etc/pacman.conf измени HoldPkg и/или IgnorePkg /etc/apt/preferences, apt-cache policy zypper mr -p правка /etc/portage/package.accept_keywords внеси строку =category/package-version
Удаление приоритета /etc/apt/preferences zypper mr -p правка /etc/portage/package.accept_keywords удали соотв. строку
Показать список приоритетов apt-cache policy или /etc/apt/preferences zypper lr -p grep -r . /etc/portage/package.accept_keywords

Верификация и восстановление

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Проверить один пакет pacman -Qk (можно добавить еще k) rpm -V debsums rpm -V equery check
Проверить все пакеты pacman -Qk (можно добавит еще k) rpm -Va debsums rpm -Va equery check
Переустановка пакета pacman -S dnf reinstall apt install --reinstall zypper install --force emerge -1O
Проверить зависимости по всей системе pacman -Dk dnf repoquery --requires apt-get check zypper verify emerge -uDN @world
Исправление сломанных зависимостей для уровня зависимостей pacman, используй pacman -Dk; для библиотек общего доступа нужен findbrokenpkgs lddd (из devtools) dnf repoquery --unsatisfied apt-get --fix-broken а затемaptitude install zypper verify revdep-rebuild
Список контрольных точек n/a dnf history list n/a
Откатить на контрольную точку n/a dnf history rollback n/a
Отмена одиночной транзакции n/a dnf history undo n/a

Использование файлов пакетов и сборка пакетов

Action Arch Red Hat/Fedora Debian/Ubuntu SLES/openSUSE Gentoo
Запросить пакет, предоставленный в командной строке pacman -Qp rpm -qp dpkg -I
Список содержимого пакета pacman -Qpl rpmls rpm -qpl dpkg -c rpm -qpl
Установить локальный пакет из файла pacman -U dnf install apt install zypper in emerge
Обновить пакет из локального файла pacman -U dnf upgrade debi emerge
Добавить локальный пакет в кеш cp package-filename /var/cache/pacman/pkg/ apt-cache add package-filename n/a cp package-filename /usr/portage/distfiles
Распаковать пакет tar -xvf rpm2cpio | cpio -vid dpkg-deb -x rpm2cpio | cpio -vid tar -jxvf
Установка / удаление с учетом зависимостей Используй ABS и выполни makepkg -seoc dnf builddep apt-get build-dep zypper si -d emerge -o
Отобразить пакет с исходниками dnf repoquery -s apt-cache showsrc n/a
Загрузить пакеты с исходниками Исп. ABS и выполни makepkg -o dnf download --source apt-get source или debcheckout zypper source-install emerge --fetchonly
Сборка из исходников makepkg -s rpmbuild -ba (обычно) или mock (в chroot) debuild rpmbuild -ba, затем build, и затем osc build ebuild или quickpkg
Проверить на проблемы с упаковкой namcap (требует namcap) rpmlint lintian rpmlint repoman
